### 1. TLB Cache: translation lookaside buffer

1.2. dTLB Cache.

### 2. Multi-level cache:
1. L1:
L1 cache can be had:
- L1.instruction cache
- L1.data cache
- L1.iTLB cache
- L1.dTLB cache
2. L2:
L2 cache can be had:
- L2.iTLB cache
- L2.dTLB cache
- L2.unified (all kind cache at L2: iTLB, dTLB, and other...)
